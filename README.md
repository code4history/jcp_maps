Japan City Plans
========

List of TMS (Tile Map Service) data based on [U.S. Army's Japan City Plans, PCL Map Collection, University of Texas Libraries](http://www.lib.utexas.edu/maps/ams/japan_city_plans/).

[テキサス大学図書館が公開している、米軍が作った敗戦直後の日本都市計画図](http://www.lib.utexas.edu/maps/ams/japan_city_plans/)をTMS (Tile Map Service)で配信するプロジェクトです。  
まだ一部の地図しかTMS化できていませんが、時間はそれなりに取るものの簡単に作れる設定ファイルで増やせますので、皆さんのご協力が公開範囲増の助けになります。  

## About

Map data in this list was made based on U.S. Army's Japan City Plans 1945-1946, PCL Map Collection, University of Texas Libraries.  
You can see them in http://www.lib.utexas.edu/maps/ams/japan_city_plans/

これらの地図データは1945-46年の、米軍の日本都市計画図をテキサス大学図書館が公開しているものを基にしています。  
こちらで公開されています。 http://www.lib.utexas.edu/maps/ams/japan_city_plans/

## Data types

Data are provided in these formats, TMS(TileJson), mbtiles and kml.  
Now using http, changing to https step by step.

データはTMS(TileJson), mbtiles と kml の形で提供しています。  
今はhttpを使っていますが、徐々にhttpsに変更いたします。
